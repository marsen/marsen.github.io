[еҜҰдҪңзӯҶиЁҳ] MacBook Terminal зҫҺеҢ–иҲҮиЁӯиЁҲ

2024жӣҙж–°

йҮҚеҜ«жӯӨж–ҮпјҢи«ӢеҸғиҖғпјҢйғЁд»ҪиіҮиЁҠд»Қжңүз”ЁйҖ”ж•…пјҢиҲҠж–ҮдёҚеҲӘ

еүҚжғ…жҸҗиҰҒ

3 е№ҙеүҚеҜ«йҒҺеҰӮдҪ•и®“ windows д№ҹжңүзҫҺзҫҺе‘Ҫд»ӨжҸҗзӨәиҰ–зӘ—,
дёҚйҒҺеңЁеҺ»е№ҙжҲ‘й–Ӣе§ӢдҪҝз”Ё MacBook Pro дәҶ, з•¶з„¶д№ҹжғіиҰҒзҫҺзҫҺзҡ„ Terminal е•Ұ.
дҪҶжҳҜйҖҷеҖӢжғіжі•дёҖзӣҙеңЁж”ҫеҝғиЈЎйқўжІ’жңүеҜҰиёҗ, з•ўз«ҹеҸӘжҳҜдёҖеҖӢжө®иӘҮзҡ„жқұиҘҝ.
дёҚйҒҺеңЁйҳІз–«жңҹй–“йҮҚж–°зңӢдәҶдёҖйҒҚ й«ҳиҰӢйҫҚзҡ„еҚіе°ҮеӨұеӮізҡ„еҸӨиҖҒжҠҖи—қ еҪұзүҮ,
йҷӨдәҶжҠҠ Vim еҶҚзҶҹжӮүдёҖйҒҚеӨ–, еҗҢжҷӮд№ҹи§ёеӢ•дәҶеҝғдёӯжө®иӘҮзҡ„йӮЈеЎҠ.
еҜҰдҪңдёҠжҜ” Windows з°Ўе–®еҫҲеӨҡ, еңЁйҖҷиЈЎзЁҚдҪңиЁҳйҢ„.

Overview

  • дёӢијү iTerm2
  • иЁӯе®ҡ iTerm2
  • е®үиЈқ oh-my-zsh
  • е®үиЈқ powerlevel10k

з¬¬дёҖжӯҘ дёӢијүдёҰе®үиЈқ iTerm2

иЁӯе®ҡ iTerm2 зҡ„еӨ–и§Җ

  1. еҫһз¶Із«ҷ iTerm2-Color-Schemes (https://github.com/mbadolato/iTerm2-Color-Schemes) дёӢијүдҪ е–ңжӯЎзҡ„й…ҚиүІж–№жЎҲпјҢдҫӢеҰӮ вҖңDimmedMonokai.itermcolorsвҖқгҖӮ
  2. й–Ӣе•ҹ iTerm2пјҢй»һж“ҠиҸңе–®ж¬„зҡ„ вҖңiTerm2вҖқпјҢйҒёж“Ү вҖңPreferencesвҖқгҖӮ
  3. еңЁеҒҸеҘҪиЁӯе®ҡиҰ–зӘ—дёӯпјҢйҒёж“Ү вҖңProfilesвҖқ йҒёй …еҚЎпјҢз„¶еҫҢйҒёж“ҮдҪ иҰҒжӣҙж”№й…ҚиүІж–№жЎҲзҡ„жңғи©ұй…ҚзҪ®ж–Үд»¶гҖӮ
  4. еңЁ вҖңColorsвҖқ йҒёй …еҚЎдёӢпјҢй»һж“Ҡ вҖңColor PresetsвҖқ жҢүйҲ•пјҢйҒёж“Ү вҖңImportвҖҰвҖқгҖӮ
  5. жүҫеҲ°еүӣжүҚдёӢијүзҡ„й…ҚиүІж–№жЎҲжӘ”жЎҲпјҢдҫӢеҰӮ вҖңDimmedMonokai.itermcolorsвҖқпјҢй»һж“Ҡ вҖңOpenвҖқгҖӮ
  6. йҒёж“Ү вҖңDimmedMonokaiвҖқ дҪңзӮәдҪ зҡ„ iTerm2 й…ҚиүІж–№жЎҲгҖӮ

Profiles иЈЎжңүжӣҙеӨҡзҡ„иЁӯе®ҡ, еӯ—еһӢгҖҒйЎҸиүІ
жҜ”еҰӮиӘӘ, иӘҝж•ҙе•ҹе§ӢиҰ–зӘ—еӨ§е°ҸиҲҮиғҢжҷҜйҖҸжҳҺеәҰ, еҸҜд»ҘеүҚеҫҖ Windows йҖІиЎҢиЁӯе®ҡ.
жӣҙеӨҡзҡ„зҙ°йғЁиЁӯе®ҡеҸҜд»ҘиҮӘиЎҢж‘ёзҙў. иЁҳеҫ—жңүдәӣж•ҲжһңйңҖиҰҒжүӢеӢ•йҮҚе•ҹ iTerm

з¬¬дәҢжӯҘ, е®үиЈқ oh-my-zsh

е®үиЈқзӣёз•¶з°Ўе–®, жӣҙеӨҡиіҮиЁҠеҸҜд»ҘеҸғиҖғ Github
йҖҷиЈЎиҰҒдәҶи§ЈеҰӮдҪ•е°Қ ~/.zshrc йҖІиЎҢз·ЁијҜ, еңЁеҫҢйқўе®үиЈқжҲ‘еҖ‘жңғйңҖиҰҒз·ЁијҜйҖҷеҖӢжӘ”жЎҲ,
еҸҜд»ҘзўәиӘҚдёҖдёӢжӘ”жЎҲе…§е®№

е®үиЈқ Powerlevel10k

еүҚеҫҖ Powerlevel10k
жҲ‘еҖ‘дҪҝз”Ё oh-my-zsh зҡ„ж–№ејҸе®үиЈқ

е®үиЈқеҫҢз·ЁијҜ ~/.zshrc иЁӯе®ҡеҸғеҰӮдёӢ:

ZSH_THEME="powerlevel10k/powerlevel10k"

д№ӢеҫҢйҮҚе•ҹ iTerm2 е°ҮжңғжңүдёҖйҖЈдёІиЁӯе®ҡе•ҸйЎҢ, дҫқз…§е–ңжӯЎзҡ„иЁӯе®ҡеҚіеҸҜ,
еҸҜд»ҘеҸғиҖғйҖҷеҖӢеҪұзүҮ,
еҰӮжһңиЁӯе®ҡе®ҢеҫҢдёҚе–ңжӯЎ, еҸҜд»Ҙеҹ·иЎҢ p10k configure йҮҚж–°иЁӯе®ҡ

е…¶е®ғ

йҖҷиЈЎиЁҳйҢ„дёҖдёӢжҲ‘жңғдҪҝз”Ёзҡ„иЁӯе®ҡ

~/.zshrc

е®үиЈқе®Ң oh-my-zsh  еҫҢеҰӮжһңжңүиіҮж–ҷеӨҫе•ҸйЎҢ, жңғеҮәзҸҫд»ҘдёӢиЁҠжҒҜ

дёҖиҲ¬дҫҶиӘӘеҸҜд»Ҙеҹ·иЎҢ compaudit | xargs chmod g-w,o-w жҢҮд»ӨеҶҚйҮҚе•ҹ iTerm2 еҚіеҸҜд»ҘжҺ’йҷӨжӯӨе•ҸйЎҢ.
еҰӮжһңз„Ўжі•жҺ’йҷӨ, еҸҲиҰәеҫ—жҜҸж¬ЎеҮәзҸҫйҖҷеҖӢиЁҠжҒҜеҫҲз…©дәәзҡ„и©ұ, и«ӢиҮі ~/.zshrc иЁӯе®ҡд»ҘдёӢиіҮиЁҠ

жҙ»з”Ё Terminal зҡ„е·ҰеҸіз©әй–“

еңЁдёҖиҲ¬зҡ„зөӮз«Ҝж©ҹдёҠ, йҡЁи‘—ж·ұе…ҘиіҮж–ҷеӨҫзөҗж§Ӣд№Ӣдёӯ,
