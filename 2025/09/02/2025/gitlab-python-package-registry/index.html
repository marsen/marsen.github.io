[еҜҰдҪңзӯҶиЁҳ] е»әз«Ӣз§Ғжңү Python Package Registry - д»Ҙ ONNX Runtime зӮәдҫӢ

еүҚжғ…жҸҗиҰҒ

еңЁй–Ӣзҷј AI жҮүз”ЁжҷӮпјҢжҲ‘еҖ‘йҒҮеҲ°дёҖдәӣзү№ж®Ҡзҡ„дҫқиіҙз®ЎзҗҶйңҖжұӮпјҢйңҖиҰҒз§Ғжңү Registry

еңЁйҖҷеҖӢе°ҲжЎҲдёӯпјҢжҲ‘еҖ‘йҒҮеҲ°дәҶе№ҫеҖӢжҢ‘жҲ°пјҡ

  1. е№іеҸ°зү№еҢ–йңҖжұӮпјҡJetson е№іеҸ°йңҖиҰҒзү№иЈҪзҡ„ onnxruntime-gpu зүҲжң¬пјҢPyPI дёҠжІ’жңүзҸҫжҲҗзҡ„
  2. зүҲжң¬дёҖиҮҙжҖ§пјҡзўәдҝқжүҖжңүз’°еўғпјҲй–ӢзҷјгҖҒжё¬и©ҰгҖҒз”ҹз”ўпјүдҪҝз”Ёе®Ңе…ЁзӣёеҗҢзҡ„дҫқиіҙзүҲжң¬
  3. е®үе…ЁжҖ§иҖғйҮҸпјҡйҒҝе…ҚдҫқиіҙеӨ–йғЁдёҚз©©е®ҡзҡ„дҫҶжәҗпјҢйҷҚдҪҺдҫӣжҮүйҸҲж”»ж“ҠйўЁйҡӘ
  4. е…§йғЁеҘ—д»¶еҲҶзҷјпјҡеңҳйҡҠй–Ӣзҷјзҡ„е…§йғЁе·Ҙе…·йңҖиҰҒжңүз®ЎйҒ“еҲҶзҷј

з§Ғжңү Registry йҒӢдҪңжөҒзЁӢ

з§Ғжңү Package Registry зҡ„йҒӢдҪңеҢ…еҗ«дёүеҖӢйҡҺж®өпјҡ

  1. еҘ—д»¶зҷјеёғйҡҺж®ө
    й–ӢзҷјиҖ… вҶ’ жҺЁйҖҒзЁӢејҸзўј вҶ’ GitLab CI/CD вҶ’ ж§Ӣе»әеҘ—д»¶ вҶ’ жҺЁйҖҒиҮі Package Registry

  2. еҘ—д»¶з®ЎзҗҶйҡҺж®ө
    Package Registry вҶҗ зү№иЈҪзүҲжң¬еҘ—д»¶пјҲеҰӮ onnxruntime-gpu for Jetsonпјү
    гҖҖгҖҖгҖҖгҖҖгҖҖгҖҖгҖҖгҖҖгҖҖвҶҗ е…§йғЁе·Ҙе…·еҘ—д»¶
    гҖҖгҖҖгҖҖгҖҖгҖҖгҖҖгҖҖгҖҖгҖҖвҶҗ е®үе…ЁеҜ©ж ёйҒҺзҡ„з¬¬дёүж–№еҘ—д»¶

  3. еҘ—д»¶дҪҝз”ЁйҡҺж®ө
    е°ҲжЎҲ pyproject.toml вҶ’ жҢҮе®ҡз§ҒжңүдҫҶжәҗ вҶ’ е®үиЈқдҫқиіҙ вҶ’ еҸ–еҫ—жӯЈзўәзүҲжң¬

йҖҷжЁЈзўәдҝқжүҖжңүз’°еўғйғҪдҪҝз”ЁдёҖиҮҙдё”еҸҜжҺ§зҡ„еҘ—д»¶зүҲжң¬гҖӮ

еҜҰдҪңжӯҘй©ҹ

зҷјдҪҲеҘ—д»¶еҲ° GitLab Package Registry

еңЁиЁӯе®ҡе’ҢдҪҝз”Ёз§Ғжңү Registry д№ӢеүҚпјҢжҲ‘еҖ‘йңҖиҰҒе…Ҳе°ҮеҘ—д»¶дёҠеӮіеҲ° GitLab Package RegistryгҖӮ

дҪҝз”Ё twine дёҠеӮіеҘ—д»¶

twine жҳҜ Python е®ҳж–№жҺЁи–Ұзҡ„еҘ—д»¶дёҠеӮіе·Ҙе…·пјҢжңғиҮӘеӢ•еҫһ wheel жӘ”жЎҲдёӯжҸҗеҸ– metadataпјҡ

еҜҰйҡӣзҜ„дҫӢпјҲд»Ҙ ONNX Runtime зӮәдҫӢпјүпјҡ

иЁӯе®ҡ GitLab Package Registry

йҰ–е…ҲпјҢеңЁ GitLab е°ҲжЎҲдёӯе•ҹз”Ё Package RegistryпјҢдёҰе»әз«Ӣ Deploy Tokenпјҡ

еңЁ GitLab е°ҲжЎҲиЁӯе®ҡдёӯе»әз«Ӣ Deploy Token, Settings вҶ’ Repository вҶ’ Deploy Tokens

ж¬Ҡйҷҗпјҡread_package_registry, write_package_registry

иЁҳдёӢ token ID е’Ң token еҖјпјҢж јејҸеҰӮдёӢпјҡ

  • Token ID: deploy-token-{ID}

  • Token: {TOKEN}

еҸ–еҫ— Project ID

жңүдёүзЁ®ж–№ејҸеҸҜд»ҘжүҫеҲ° GitLab Project IDпјҡ

еҫһе°ҲжЎҲйҰ–й Ғпјҡ

йҖІе…ҘдҪ зҡ„ GitLab е°ҲжЎҲйҰ–й Ғ
Project ID жңғйЎҜзӨәеңЁе°ҲжЎҲеҗҚзЁұдёӢж–№
дҫӢеҰӮпјҡProject ID: 12345678

еҫһе°ҲжЎҲиЁӯе®ҡй Ғйқўпјҡ

йҖІе…Ҙ Settings вҶ’ General
еңЁжңҖдёҠж–№зҡ„ вҖңGeneral project settingsвҖқ еҚҖеЎҠ
еҸҜд»ҘзңӢеҲ° Project ID

еҫһ GitLab APIпјҡ

еҰӮжһңдҪ еңЁ CI/CD pipeline дёӯпјҢеҸҜд»ҘзӣҙжҺҘдҪҝз”Ёз’°еўғи®Ҡж•ё $CI_PROJECT_ID
йҖҷеҖӢи®Ҡж•ёжңғиҮӘеӢ•её¶е…Ҙз•¶еүҚе°ҲжЎҲзҡ„ ID

й…ҚзҪ®е°ҲжЎҲзҡ„дҫқиіҙз®ЎзҗҶ

еңЁ pyproject.toml дёӯиЁӯе®ҡз§Ғжңү registryпјҡ

йҖҷиЈЎзҡ„й—ңйҚөжҳҜдҪҝз”Ёжўқд»¶дҫқиіҙпјҢж №ж“ҡдёҚеҗҢе№іеҸ°е®үиЈқдёҚеҗҢзүҲжң¬зҡ„еҘ—д»¶гҖӮ

й–ӢзҷјиҖ…дҪҝз”Ёз§Ғжңү Registry

ж №ж“ҡе°ҲжЎҲзҡ„е®үе…Ёзӯ–з•ҘпјҢRD жңүе№ҫзЁ®ж–№ејҸеӯҳеҸ–з§Ғжңү registryпјҡ

йҒёй … 1пјҡToken е…§еөҢеңЁ pyproject.tomlпјҲз°Ўе–®дҪҶдёҚе®үе…Ёпјү

еҰӮжһң pyproject.toml дёӯе·Із¶“еҢ…еҗ«е®Ңж•ҙзҡ„иӘҚиӯү URLпјҡ

йҒёй … 2пјҡдҪҝз”Ёз’°еўғи®Ҡж•ёпјҲжҺЁи–Ұпјү

еңЁ pyproject.toml дёӯдҪҝз”ЁдҪ”дҪҚз¬Ұпјҡ

RD йңҖиҰҒиЁӯе®ҡз’°еўғи®Ҡж•ёпјҡ

йҒёй … 3пјҡдҪҝз”ЁиӘҚиӯүжӘ”жЎҲ

иЁӯе®ҡ pip жҲ– uv зҡ„иӘҚиӯүжӘ”жЎҲпјҡ

йҒёй … 4пјҡдјҒжҘӯе…§з¶ІеӯҳеҸ–

еҰӮжһңдҪҝз”ЁдјҒжҘӯе…§з¶ІжҲ– VPNпјҡ

CI/CD дёӯеӯҳеҸ–з§Ғжңү Registry

е°ҲжЎҲеңЁ GitLab CI/CD дёӯйңҖиҰҒеӯҳеҸ–з§Ғжңү Registry жҷӮпјҢжңүд»ҘдёӢе№ҫзЁ®ж–№ејҸпјҡ

ж–№жі• 1пјҡдҪҝз”Ё CI_JOB_TOKENпјҲжҺЁи–Ұпјү

ж–№жі• 2пјҡдҪҝз”Ё Deploy Token

еңЁ CI/CD и®Ҡж•ёдёӯиЁӯе®ҡ GITLAB_DEPLOY_TOKENпјҡ

ж–№жі• 3пјҡй…ҚеҗҲ uv дҪҝз”Ё

еёёиҰӢе•ҸйЎҢиҲҮи§Јжұәж–№жЎҲ

Deploy Token ж¬ҠйҷҗдёҚи¶іпјҹ

зўәдҝқ Deploy Token жңү read_package_registry е’Ң write_package_registry ж¬ҠйҷҗгҖӮ

еҘ—д»¶зүҲжң¬иЎқзӘҒпјҹ

дҪҝз”Ё uv зҡ„ resolution markers еҠҹиғҪпјҢжҳҺзўәжҢҮе®ҡзүҲжң¬и§Јжһҗзӯ–з•ҘгҖӮ

Registry URL ж јејҸйҢҜиӘӨпјҹ

жіЁж„Ҹе°Ү {PROJECT_ID} жӣҝжҸӣзӮәеҜҰйҡӣзҡ„е°ҲжЎҲ IDгҖӮ

е°Ҹзөҗ

йҖҸйҒҺе»әз«Ӣз§Ғжңү Package RegistryпјҢжҲ‘еҖ‘жҲҗеҠҹи§ЈжұәдәҶпјҡ

  • зү№иЈҪзүҲжң¬з®ЎзҗҶпјҡJetson е№іеҸ°зҡ„зү№ж®Ҡ ONNX Runtime зүҲжң¬йңҖжұӮ
  • дҫқиіҙдёҖиҮҙжҖ§пјҡжүҖжңүз’°еўғдҪҝз”ЁзӣёеҗҢзүҲжң¬зҡ„еҘ—д»¶
  • е®үе…ЁжҖ§жҸҗеҚҮпјҡжёӣе°‘е°ҚеӨ–йғЁдҫҶжәҗзҡ„дҫқиіҙ
  • еӯҳеҸ–дҫҝеҲ©жҖ§пјҡй–ӢзҷјиҖ…е’Ң CI/CD йғҪиғҪиј•й¬ҶеӯҳеҸ–з§ҒжңүеҘ—д»¶
